# Uniswap Monitor Bot

A go-ethereum bot that tracks and monitors `Swap` events on Uniswap USDC2 pool.

## To run Locally

1. run `go mod vendor`
2. run `go run main.go`

(or)

1. run `./uniswap-monitor-bot` to directly run the executable
